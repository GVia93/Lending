## 🚀 Как проверить SEO

### Онлайн инструменты:
1. **Google PageSpeed Insights** - скорость загрузки
2. **Schema Markup Validator** - проверка JSON-LD
3. **Open Graph Debugger** (Facebook)
4. **Twitter Card Validator**
5. **Яндекс.Вебмастер** - проверка robots.txt и sitemap

### Команды для проверки:
```bash
# Проверить robots.txt
curl https://m-stroy.ru/robots.txt

# Проверить sitemap.xml
curl https://m-stroy.ru/sitemap.xml

# Проверить meta теги
curl -s https://m-stroy.ru | grep -i "meta name"
```
